# Tasty

Tasty a food logger with MySQL, Node, Express, Handlebars and a homemade ORM (yum!). 
Heroku Deployed Link: https://tastyyy.herokuapp.com/

<br>



## Get Started
Please enter the following command in your terminal:
```
$ npm i
```
